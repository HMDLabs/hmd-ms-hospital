.. hmd_lang_hospital.device

hmd_lang_hospital.device
=============================================

**Metatype**: noun


Attributes
++++++++++


* **code (string)** : A medical code for the entity, taken from a controlled vocabulary or ontology such as ICD-9, DiseasesDB, MeSH, SNOMED-CT, RxNorm, etc.

* **legal_status (string)** : The drug or supplement's legal status, including any controlled substance schedules that apply.

* **alternate_name (string)** : An alias for the item.

* **description (string)** : A description of the item.

* **name (string)** (required): The name of the item.

* **serial_number (string)** (required): The serial number for the device.
